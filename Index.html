Components of a CPU

  • Control Unit: Fetches and decodes instructions.
  • ALU: Performs math and logical operations.
  • Registers: Small fast storage inside the CPU.
  • Cache: Stores frequently used data for faster access.

What Affects CPU Performance

  • Clock speed (GHz)
  • Number of cores
  • Cache size
  • Processor architecture
  • Temperature and power limits

Embedded Computers

Embedded computers are built for one specific task inside a larger device. They are found in cars, appliances, medical machines and traffic lights. They usually use microcontrollers that combine processor, memory and I/O on one chip.

Advantages of Embedded Systems

  • More energy efficient
  • Smaller and cheaper
  • More reliable
  • Less complex
  • Predictable real-time performance
